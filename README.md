# UR Aerotech Website

This is a responsive aviation company website built using **HTML** and **Tailwind CSS**.

The project represents a modern website for an aircraft repair and aviation solutions company called **UR Aerotech**.

## 🚀 Features

* Responsive Navbar
* Hero Section with Background Image
* Solutions Cards Section
* Sign-In Modal Popup
* Responsive Design
* Modern UI using Tailwind CSS
* Footer with company information

## 🛠 Technologies Used

* HTML5
* Tailwind CSS
* JavaScript

## 📂 Project Structure

```
uraerotech

     ├── aircartfs.jpeg
     ├── airoplan.jpg

     └── home img.jpg
```

## 💻 How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Run **index.html** in your browser
